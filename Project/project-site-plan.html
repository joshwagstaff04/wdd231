High Performance Living Toolkit — Site Plan

Individual Project Site Plan for WDD 231 • Planning the home page, habit library, and tools/daily planner experience.

Site Name

Site Name: High Performance Living Toolkit

I chose this name because it makes it clear the site is about more than general wellness. It focuses on living at a higher daily performance using practical, habit-based tools. The word “Toolkit” helps visitors expect simple, usable habits they can try right away.

Site Purpose

This site is meant to help people improve their daily energy, focus, sleep quality, and overall productivity using simple, science-backed habits and easy-to-use tools.

The site will include:

  • An overview of the core pillars of high performance living: energy, focus, environment, and everyday habits.
  • A dynamic habit library where users can explore specific, research-informed habits related to energy, mood, and productivity and save their favorites.
  • A tools and daily planner page that helps users set up a simple “daily reset” plan and review their saved habits in one place.

Scenarios

These scenarios are examples of what visitors to the High Performance Living Toolkit might be asking. I’ll use them to guide which content and features I include.

  • Scenario 1: “I feel drained and unfocused by mid-afternoon. What small, science-backed habits can I use to improve my energy and focus without relying on more caffeine or energy drinks?”
  • Scenario 2: “I want to build a simple daily routine that supports better sleep, movement, and deep work time. Is there a planner or template that can help me design a high-performance day?”
  • Scenario 3: “I’ve tried complicated productivity systems before and didn’t stick with them. Can I quickly bookmark a few key habits and revisit them whenever I need a reset?”

Color Scheme

The color scheme is meant to feel calm and focused but still energetic. Dark blues create a grounded, night-time feel that ties into focus and sleep, while the bright green accent suggests growth, progress, and “high performance.”

Primary Color
#0F172A — Deep Navy
Used for the main page background and header background to create a focused, immersive environment.
Surface Color
#111827 — Surface Slate
Used for cards, sections, and content containers to provide subtle separation from the main background.
Accent Color
#22C55E — Performance Green
Used for headings, buttons, key highlights, and interactive elements like “Favorite” controls to emphasize action and success.
Muted / Supporting Color
#9CA3AF — Muted Gray
Used for secondary text, helper descriptions, borders, and less important details to keep focus on core content.

This site plan document uses the same colors for its background, cards, headings, and accent elements.

Typography

I’m using one font for headings and one for body text to keep things clean and easy to read. Both are common web fonts and work well for accessibility.

Heading Font — Poppins

Font: Poppins, sans-serif
Usage: Main headings (<h1>, <h2>) and key section titles across the site, including the hero heading and major section titles on the Home, Habit Library, and Tools & Daily Planner pages. Poppins has a slightly rounded, modern look that fits the “high performance but friendly” vibe I’m going for.

Body Font — Inter

Font: Inter, system-ui, sans-serif
Usage: All body copy, paragraph text, labels, form inputs, and habit descriptions. Inter is easy to read on screens and works well for longer text like habit descriptions and tool instructions.

Both fonts are loaded with Google Fonts in this document so it matches how I plan to use them on the final site.
